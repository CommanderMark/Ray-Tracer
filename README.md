# Ray Tracer
Little more than a basic ray calculator that spits out an image.

Supports:
* Phong illumination model to an extent.
* Shadows.
* Spheres.
* Planes.
* Triangles.
* OBJ Meshes (taken from [Bly7/OBJ-Loader][obj-loader]).
